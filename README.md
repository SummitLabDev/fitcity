## Tech Stack

- **Frontend:** React 18 with Vite
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Routing:** React Router
- **Hosting:** Cloudflare Pages
- **Language:** Dutch (primary market)
